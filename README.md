# Guía de StoneScript en español

Una **guía completa de StoneScript** (el lenguaje de **Stone Story RPG**) explicada de
forma fácil y entendible, con la idea de **preguntas y respuestas**.

## 📖 Guía de StoneScript (empieza aquí)

¿Quieres aprender a programar a tu personaje? La guía explica **todo** el lenguaje paso a
paso, con la idea de **preguntas y respuestas** (`?` = si, `:?` = si no pero si, `:` = si
no), condiciones anidadas, funciones, variables, bucles, interfaz y mucho más.
